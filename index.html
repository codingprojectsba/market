How it works

Users submit a lower 25%, median, and upper 75% estimate for a numeric question. The system converts each prediction into a smooth distribution and aggregates predictions using stake-weighted averaging.

Current version: each numeric question has its own minimum, maximum, and unit. Yes/no questions will be handled separately.
